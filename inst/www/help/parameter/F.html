The F-statistic is the ratio of variance explained by the model to the variance unexplained by the model.

Formula:

$F = { \sigma_{explained}^2 \over \sigma_{unexplained}^2 } $
or
$F = { between-group variability \over within-group variability } $

Suppose we use mean ($\bar{x}$) to model the data, explained variance is given by

$\sigma_{explained}^2 = { { \sum_{i} n_{i} (\bar{x}_{i} - \bar{x})^2 } \over {K - 1} }$

Where $\bar{x}_{i}$ denotes the sample mean in the ith group, $n_{i}$ is the number of observations in the ith group,$\bar{x}$ denotes the overall mean of the data, and $K$ denotes the number of groups.

And, unexplained variance is given by :
$\sigma_{unexplained}^2 = { { \sum_{ij} ( x_{ij} - \bar{x}_{i} )^2 } \over {N - K} }$

Where $x_{ij}$ is the jth observation in the ith out of $K$ groups and $N$ is the overall sample size.

Legend:

  • $\sigma_1^2 = $ Variance One
  • $\sigma_2^2 = $ Variance Two
  • $\sigma^2 = $ Variance
  • $x = $ Values given in a set of data
  • $\bar{x} = $ Mean of the data
  • $n = $ Total number of values
