V statistics (also known as Z statistics) is used in Wilcoxon signed-rank test.

Formula:

$ Z = {(W - 0.5 -N * (N+1) \div 4 ) } \div {\sqrt{ { ( N * (N + 1) * (2 * N + 1 ) ) } \div 24} } $

Legend:

  • $N = $ Total number of pairs
  • $W = $ W is larger of $W^+$ or $W^-$

Procedure:

Rank the differences without regard to the sign of the difference (i.e., rank order the absolute differences). Ignore all zero differences (i.e., pairs with equal members, x=y). Affix the original signs to the rank numbers. All pairs with equal absolute differences (ties) get the same rank: all are ranked with the mean of the rank numbers that would have been assigned if they would have been different. Sum all positve ranks (W+) and all negative ranks (W-) and determine the total number of pairs (N).
